## 1. 目的

规范住院患者出院流程，确保出院手续高效、准确办理，保障医疗文书完整，做好出院指导与随访衔接，提升患者满意度。

## 2. 适用范围

适用于所有经主管医师评估达到出院标准的住院患者。

## 3. 术语与定义

| 术语 | 定义 |
|------|------|
| 出院医嘱 | 主管医师下达的允许患者离院的医疗指令 |
| 出院带药 | 患者出院时携带的后续治疗药物 |
| 出院小结 | 记录住院诊疗经过、出院诊断及出院后注意事项的医疗文书 |
| 医保结算 | 按医保政策进行费用审核与报销的结算流程 |
| 随访 | 出院后对患者病情恢复情况进行跟踪的医疗服务 |

## 4. 职责分工

### 4.1 主管医师
- 评估出院标准，下达出院医嘱
- 完成出院病历及出院小结
- 开具出院带药处方
- 进行出院医患沟通与指导

### 4.2 病区护士站
- 执行出院医嘱，停止相关治疗
- 进行出院宣教
- 整理出院病历资料
- 撤销床头卡，通知住院处

### 4.3 住院处
- 办理出院费用结算
- 打印出院结算单据
- 处理医保报销
- 归档出院信息

### 4.4 药房
- 审核出院带药处方
- 发放出院带药
- 交代用药注意事项

## 5. 操作流程

### 5.1 出院医嘱流程

```
医师评估出院标准 → 下达出院医嘱 → 通知护士站及患者/家属
→ 交代出院后注意事项 → 完成出院病历
```

**详细步骤：**

1. **出院评估**：
   - 确认患者病情稳定，符合出院标准
   - 确认检查检验结果回报完整
   - 确认手术/操作后恢复情况良好（如适用）

2. **医嘱下达**：
   - 下达出院医嘱（注明出院日期）
   - 开具出院带药处方
   - 开具出院诊断证明（如需）
   - 告知护士站预计出院时间

3. **病历完成**：
   - 完成出院记录
   - 完成出院小结
   - 完善各项知情同意书（如未签署）
   - 确保病历质量符合要求

### 5.2 病区执行流程

```
收到出院医嘱 → 停止相关治疗 → 整理病历资料 → 出院宣教
→ 通知住院处 → 撤销床单位 → 患者离区
```

**详细步骤：**

1. **医嘱执行**：
   - 停止长期医嘱与临时医嘱
   - 停止生命体征监测
   - 撤销输液、吸氧等治疗
   - 核对医嘱执行完整性

2. **病历整理**：
   - 收集各项检查检验报告单
   - 整理体温单、护理记录
   - 核对病历资料完整性
   - 送交住院处/病案室归档

3. **出院宣教**：
   - 用药指导：带药名称、用法、注意事项
   - 饮食指导：出院后饮食建议
   - 活动指导：休息与活动建议
   - 伤口护理：换药时间、注意事项（如适用）
   - 复诊安排：复诊时间、地点、需携带资料
   - 紧急情况处理：何种情况需立即就医

4. **床单位处理**：
   - 撤销床头卡
   - 清理床单位，更换床单被褥
   - 检查有无遗留物品
   - 通知住院处该床位可用

### 5.3 费用结算流程

```
患者/家属持出院通知到住院处 → 核对费用明细 → 医保审核（如适用）
→ 结算费用 → 收取/退还余额 → 打印结算单据 → 发放病历资料
```

**详细步骤：**

1. **费用核对**：
   - 核对住院期间所有费用明细
   - 确认用药、检查、治疗、床位等费用
   - 核对出院带药费用
   - 处理患者疑问

2. **医保结算**（如适用）：
   - 核验医保信息
   - 上传医保结算数据
   - 医保系统审核
   - 计算医保报销金额与自付金额

3. **费用收取/退还**：
   - 计算应收/应退金额
   - 收取剩余费用或退还多交押金
   - 开具正式发票/收据

4. **资料发放**：
   - 出院结算单
   - 费用明细清单
   - 出院诊断证明（如需）
   - 病历复印件（如需）
   - 医保报销材料（如需）

### 5.4 药品发放流程

```
患者持出院带药处方到药房 → 药师审核处方 → 核对药品
→ 发放药品 → 交代用药注意事项 → 签字确认
```

**详细步骤：**

1. **处方审核**：
   - 核对患者身份与处方信息
   - 审核药品合理性（剂量、用法、相互作用）
   - 确认出院带药符合医保政策

2. **药品发放**：
   - 按处方调配药品
   - 核对药品名称、规格、数量
   - 标注用法用量
   - 发药并请患者/家属签字

3. **用药交代**：
   - 口头交代用法用量
   - 说明注意事项与不良反应
   - 告知药品储存条件
   - 确认患者/家属理解

## 6. 特殊情况处理

### 6.1 自动出院（患者要求提前离院）
- 充分告知风险
- 签署知情自动出院书
- 完成已发生费用结算
- 做好病历记录与随访登记

### 6.2 转院出院
- 完善转院手续
- 与接收医院做好病情交接
- 准备完整病历资料副本
- 安排转运方式

### 6.3 死亡出院
- 完善死亡病历
- 开具死亡医学证明
- 做好家属沟通与安抚
- 遗体交接按规定办理
- 病历归档保存

### 6.4 欠费出院
- 上报医务科/财务科
- 签署欠费协议
- 做好催收登记
- 按医院相关规定处理

### 6.5 医保特殊结算
- 异地医保：提前办理异地就医备案
- 商业保险：提供完整费用清单与诊断证明
- 工伤保险：提供工伤认定材料
- 特殊药品：确认医保特药审批状态

## 7. 出院宣教内容要点

| 类别 | 内容 |
|------|------|
| 用药 | 药品名称、剂量、频次、疗程、注意事项 |
| 饮食 | 饮食禁忌、营养建议、特殊饮食要求 |
| 活动 | 休息时间、活动限制、康复锻炼 |
| 伤口 | 换药时间、观察要点、洗澡注意事项 |
| 复诊 | 复诊时间、地点、需携带资料 |
| 紧急 | 需立即就医的症状与体征 |
| 随访 | 随访方式、随访时间 |

## 8. 服务质量要求

| 项目 | 标准 |
|------|------|
| 出院医嘱下达至患者离区 | ≤ 4 小时 |
| 出院结算时间 | ≤ 15 分钟 |
| 出院宣教完成率 | 100% |
| 病历归档及时率 | ≤ 24 小时 |
| 出院带药交代完成率 | 100% |
| 患者满意度 | ≥ 90% |
| 出院诊断证明出具 | 当日完成 |

## 9. 交接班管理

1. 交接当日出院患者清单
2. 交接未完成出院手续的患者
3. 交接欠费患者信息
4. 交接待归档病历
5. 交接未发放的检查报告
6. 双方签字确认

## 10. 信息安全管理

- 患者出院信息严格保密
- 病历资料按规定归档保管
- 结算信息不得外泄
- 医保数据按规定上传与保存
- 出院患者个人信息保护
