## 1. 目的

规范住院患者入院流程，确保患者安全、有序办理入院手续，保障医疗文书完整，提升入院服务效率与患者满意度。

## 2. 适用范围

适用于所有经门诊或急诊医生开具住院证后，需办理住院手续的患者。

## 3. 术语与定义

| 术语 | 定义 |
|------|------|
| 住院证 | 由主治及以上医师开具的住院通知凭证 |
| 入院登记 | 患者信息录入住院管理系统的过程 |
| 床位协调 | 根据病情和科室床位情况进行床位分配 |
| 预住院 | 患者已办理手续但暂未安排正式床位的等待状态 |
| 入院评估 | 入院时对患者病情、护理需求的综合评估 |

## 4. 职责分工

### 4.1 住院处工作人员
- 核验患者身份及住院证件
- 办理入院登记与押金收取
- 录入患者基本信息至住院系统
- 发放入院须知及相关资料

### 4.2 病区护士站
- 确认床位并通知住院处
- 接收患者并安排床位
- 进行入院护理评估
- 入院宣教及环境介绍

### 4.3 主管/值班医师
- 审核入院诊断
- 开具入院医嘱
- 首次病程记录
- 与患者/家属沟通病情及住院方案

### 4.4 床位协调中心（如有）
- 统筹全院床位资源
- 协调跨科收治
- 处理加床与转科需求

## 5. 操作流程

### 5.1 入院登记流程

```
患者持住院证到住院处 → 核验身份及证件 → 录入住院信息
→ 收取押金 → 打印入院通知单 → 引导至病区
```

**详细步骤：**

1. **证件核验**：
   - 查验住院证（需有主治及以上医师签名）
   - 核对患者身份证、医保卡
   - 确认住院证信息与证件一致

2. **信息录入**：
   - 录入患者基本信息（姓名、性别、年龄、身份证号、联系方式）
   - 录入联系人/紧急联系人信息
   - 关联门诊/急诊就诊记录
   - 录入诊断信息

3. **费用处理**：
   - 告知预交押金金额
   - 支持现金/刷卡/移动支付/医保预登记
   - 开具押金收据
   - 告知费用查询方式

4. **发放资料**：
   - 入院须知（含住院规则、探视制度、膳食安排等）
   - 患者权利与义务告知书
   - 入院通知单（注明病区、床位）

5. **引导入区**：
   - 告知病区位置与交通指引
   - 特殊患者（轮椅、平车）协助转运

### 5.2 病区接收流程

```
收到入院通知 → 确认床位 → 准备床单位 → 患者到达 → 核对身份
→ 安置患者 → 入院宣教 → 通知医师
```

**详细步骤：**

1. **床位准备**：
   - 护士站确认入院通知
   - 检查床位状态，确认可用
   - 备好床单、被褥、病历夹等
   - 必要时准备心电监护、吸氧等设备

2. **患者接收**：
   - 核对患者身份（至少使用两种方式：姓名+住院号/身份证）
   - 核对住院证与入院通知单
   - 协助患者更换病号服
   - 协助患者安置于床位

3. **入院宣教**：
   - 介绍病区环境（护士站、医生办公室、洗手间、开水间等）
   - 介绍呼叫铃使用方法
   - 告知作息时间、探视制度
   - 告知饮食安排及注意事项
   - 介绍主管医生和责任护士

4. **护理评估**：
   - 生命体征测量（T、P、R、BP、SpO2）
   - 过敏史询问与记录
   - 跌倒/坠床风险评估
   - 压疮风险评估
   - 营养状况评估
   - 心理状态评估
   - 疼痛评估
   - 填写入院护理评估单

5. **通知医师**：
   - 通知主管医生/值班医生患者已到
   - 转交住院病历及检查资料

### 5.3 医师接诊流程

```
收到通知 → 查看病历资料 → 诊察患者 → 完善入院诊断
→ 开具入院医嘱 → 首次病程记录 → 与患者/家属沟通
```

**详细步骤：**

1. **病历审阅**：
   - 查阅门诊/急诊病历
   - 了解既往病史、用药史、过敏史
   - 查阅已有检查检验结果

2. **患者诊察**：
   - 体格检查
   - 确认或修正诊断
   - 评估住院必要性与预期住院天数

3. **医嘱开具**：
   - 护理级别
   - 饮食类型
   - 检查检验医嘱
   - 治疗方案
   - 用药医嘱

4. **病历书写**：
   - 24小时内完成首次病程记录
   - 完善入院记录
   - 知情同意书签署（如需手术/特殊检查）

5. **医患沟通**：
   - 告知患者/家属诊断与治疗方案
   - 说明可能的风险与替代方案
   - 解答患者疑问
   - 签署知情同意文件

## 6. 特殊情况处理

### 6.1 急诊入院
- 急诊住院证优先处理，开通绿色通道
- 先入院后补办手续
- 急诊医师与病房做好交接

### 6.2 床位紧张
- 启动床位协调机制
- 优先保障急危重症患者
- 预住院患者做好登记与随访
- 启动加床方案时确保安全与质量

### 6.3 身份不明患者
- 先行救治，同步报告医务科
- 暂用"无名氏"登记，附体貌特征描述
- 联系公安部门核实身份
- 费用按医院相关规定处理

### 6.4 医保患者入院
- 核验医保卡有效性
- 确认医保类型与定点资格
- 办理医保入院登记
- 告知医保相关政策与自费项目

### 6.5 患者拒绝住院
- 充分告知风险
- 签署知情拒绝住院书
- 做好登记与随访
- 必要时报告医务科

## 7. 服务质量要求

| 项目 | 标准 |
|------|------|
| 入院登记时间 | ≤ 10 分钟 |
| 病区接收时间 | ≤ 30 分钟（接到通知后） |
| 医师首次诊察 | ≤ 2 小时（入区后） |
| 首次病程记录 | ≤ 24 小时 |
| 护理评估完成 | ≤ 8 小时 |
| 患者满意度 | ≥ 90% |
| 身份核对准确率 | 100% |

## 8. 交接班管理

1. 交接当日入院患者清单
2. 交接待入院（预约）患者信息
3. 交接押金收缴情况
4. 交接系统运行状态及异常
5. 交接未完成事项
6. 双方签字确认

## 9. 信息安全管理

- 患者个人信息严格保密
- 住院系统账号专人专用
- 病历资料按规定归档保管
- 电子病历系统操作留痕
- 转科/转院患者信息完整交接
