## 1. 目的

规范门诊挂号服务流程，确保患者快速、准确完成挂号，保障门诊就诊秩序，提升患者就医体验。

## 2. 适用范围

适用于医院门诊所有挂号渠道（窗口挂号、自助机挂号、线上预约挂号）的业务操作与管理。

## 3. 术语与定义

| 术语 | 定义 |
|------|------|
| 首诊患者 | 首次到本院就诊、尚未建立就诊档案的患者 |
| 复诊患者 | 已在本院建立就诊档案、再次前来就诊的患者 |
| 预约挂号 | 通过线上平台或电话提前预约就诊时段的挂号方式 |
| 爽约 | 已预约但未按时到院取号就诊的行为 |

## 4. 职责分工

### 4.1 挂号处工作人员
- 核实患者身份信息，准确录入挂号系统
- 指导患者选择正确科室与医生
- 告知就诊流程及注意事项
- 处理挂号、退号、改号等业务

### 4.2 导诊护士
- 引导患者到正确挂号窗口或自助机
- 解答患者关于科室分布、就诊流程的咨询
- 协助特殊患者（老年人、残疾人等）完成挂号

### 4.3 信息科
- 保障挂号系统稳定运行
- 及时处理系统故障
- 定期维护号源数据

## 5. 操作流程

### 5.1 窗口挂号流程

```
患者到达 → 出示证件/就诊卡 → 核实身份信息 → 选择科室/医生
→ 确认挂号信息 → 收费 → 打印挂号凭条 → 告知就诊地点与注意事项
```

**详细步骤：**

1. **迎接患者**：主动问候，询问就诊需求
2. **身份核实**：
   - 首诊患者：查验身份证件，填写基本信息，办理就诊卡
   - 复诊患者：读取就诊卡或医保卡，调取患者信息
3. **科室选择**：
   - 根据患者主诉推荐合适科室
   - 如不确定，引导至导诊台分诊
4. **医生选择**：告知当日出诊医生信息，尊重患者选择
5. **信息确认**：复述挂号信息（科室、医生、费用），请患者确认
6. **收费处理**：
   - 现金/刷卡/移动支付/医保结算
   - 打印收费凭证
7. **发放凭条**：打印挂号凭条，标明科室、楼层、诊室、序号、就诊时段
8. **告知指引**：口头告知就诊地点，必要时指引方向或提供楼层指引图

### 5.2 自助机挂号流程

```
患者到达自助机 → 点击"挂号" → 选择证件类型并读取 → 选择科室/医生
→ 选择时段 → 确认信息 → 支付 → 打印凭条
```

**工作人员职责：**
- 巡视自助机区域，主动协助操作困难的患者
- 及时补充打印纸等耗材
- 处理自助机故障报修

### 5.3 线上预约挂号流程

```
患者线上预约 → 收到预约确认通知 → 按预约时段到院 → 自助机/窗口取号
→ 等候就诊
```

**注意事项：**
- 告知患者预约后需在规定时间内取号，超时自动取消
- 爽约累计达到规定次数，限制一定期限内的预约资格
- 线上取消需提前一定时间，否则按爽约处理

### 5.4 退号/改号流程

1. 患者提出退号/改号需求
2. 核实挂号信息及退号原因
3. 系统操作退号或改号
4. 退费处理（原路退回或现金退还）
5. 如改号，重新打印挂号凭条

## 6. 特殊情况处理

### 6.1 系统故障
- 立即报告信息科，记录故障时间
- 启用手工挂号（备用登记本）
- 告知患者预计恢复时间，安抚情绪
- 系统恢复后及时补录数据

### 6.2 急诊患者
- 立即引导至急诊科，优先处理
- 不能因挂号手续延误急诊救治
- 做好急诊分流登记

### 6.3 医保患者
- 核验医保卡有效性
- 确认医保类型（城镇职工/居民/新农合等）
- 按医保政策执行挂号费减免

### 6.4 无证/信息不全患者
- 先行人工登记，引导补办证件
- 建议办理就诊卡，方便后续就诊

### 6.5 投诉与纠纷
- 耐心倾听，不与患者争执
- 无法当场解决的，记录问题并上报门诊办
- 重大纠纷立即报告值班领导

## 7. 服务质量要求

| 项目 | 标准 |
|------|------|
| 挂号准确率 | ≥ 99.5% |
| 单人次挂号时间 | ≤ 3 分钟 |
| 患者等候时间 | 窗口 ≤ 10 分钟，自助机 ≤ 5 分钟 |
| 投诉处理响应 | 24 小时内回复 |
| 服务态度 | 使用文明用语，禁止推诿、冷漠 |

## 8. 交接班管理

1. 交接前清点现金、票据、就诊卡
2. 交接未处理完的特殊事项
3. 交接系统运行状态
4. 双方签字确认
5. 做好交接班记录

## 9. 信息安全管理

- 患者个人信息严格保密，不得外泄
- 挂号系统账号专人专用，定期更换密码
- 严禁在非授权设备上登录挂号系统
- 患者病历资料按规定归档保管
